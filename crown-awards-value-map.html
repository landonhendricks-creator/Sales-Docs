Business Objectives

  • Maintain or improve granular reporting — Jonathan selected Zima specifically for deep reporting; new platform must match or exceed. Kenny and Tatiana must be able to run reports independently without IT help

Challenges & Pain Points
  • Philippines + New York on different license types in Zima — agents cannot call each other directly without going through a queue. Tatiana called this her "biggest hurdle." Must be resolved in new platform.
  • PCI redaction requires a manual API pipeline in Zima — pull recording, redact, delete, reinstall. Crown takes phone payments constantly (mostly sales calls). This is a daily operational burden.
  • Zima's AI/WFM was third-party and failed — speech analytics and WFM were removed from contract for non-performance. Jonathan explicitly asked: show only 8x8-native features or clearly label 3rd party.
  • Softphone reliability issues — CC agents on browser-based softphones; Tatiana mentioned "issues" with them. Wants flexibility for hard phone, softphone, or mobile options.
  • Live chat is siloed — separate LiveChat vendor, separate contract. Both contracts ending December 2026. Opportunity to consolidate everything under 8x8.
  • Forecasting/scheduling is manual — busy seasons (Apr–Jun) require different staffing. No WFM tool currently since Zima's failed. Need forecasting capability without overpaying for enterprise WFM.
Strategies
  • Deploy 8x8 UCaaS + CCaaS — all locations (NY, Philippines, Belize) on same license tier; eliminates cross-location call restrictions that plagued Zima deployment
  • Enable QMSA (AI speech analytics) for 100% automated QA — every call scored, categorized, and sentiment-mapped without manual agent intervention
  • Implement Call Cabinet / Smarsh for PCI-compliant recording + redaction — fully integrated with 8x8; no custom API build required. Credit card data automatically redacted on recording.
  • Consolidate LiveChat under 8x8 omnichannel — web chat, SMS (two-way), and voice in one contact center platform and reporting view
  • Leverage built-in WFM (acquired, included with CC licenses) for seasonal forecasting — no third-party contract, no additional cost; purpose-built for Crown's use case
  • Confirm Grandstream GRP-2613 compatibility — office/non-CC users keep existing desk phones; avoid hardware refresh cost
